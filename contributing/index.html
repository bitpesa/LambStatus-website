Contributing

Thank you for seeing this page! LambStatus is still young project and any contributions are very welcome! For example:

  • Report a bug, a feature request, a design issue and even typo at GitHub Issues.

  • Open a first pull request.

    • Good issues for first-time contributors have the label good first issue.
    • Here is the document to set up the development environment.
